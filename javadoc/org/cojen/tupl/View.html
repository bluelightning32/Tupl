Interface View

  • All Known Subinterfaces:

    public interface View
    Mapping of keys to values, in no particular order. Subclasses and implementations may specify an explicit ordering.

      • lockShared

        LockResult lockShared(Transaction txn,
                    byte[] key)
                      throws LockFailureException,
                             ViewConstraintException
        Explicitly acquire a shared lock for the given key, denying exclusive locks. Lock is retained until the end of the transaction or scope.

        Transactions acquire locks automatically, and so use of this method is not required. Ownership of the key instance is transferred, and so the key must not be modified after calling this method.

        Parameters:
        key - non-null key to lock; instance is not cloned
        Returns:
        ACQUIRED, OWNED_SHARED, OWNED_UPGRADABLE, or OWNED_EXCLUSIVE
        Throws:
        IllegalStateException - if too many shared locks
        LockFailureException - if interrupted or timed out
        DeadlockException - if deadlock was detected after waiting full timeout
        ViewConstraintException - if key is not allowed

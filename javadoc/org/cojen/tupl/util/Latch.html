Class Latch

  • All Implemented Interfaces:
    Serializable

    public class Latch
extends AbstractQueuedSynchronizer
    Non-reentrant read/write latch, using unfair acquisition. Implementation also does not track thread ownership or check for illegal usage. As a result, it typically outperforms ReentrantLock and Java synchronization.

    Brian S O'Neill

    public class DeadlockException
extends LockTimeoutException
    Thrown when a lock request by a transaction timed out due to a deadlock. Deadlocks can be prevented by locking records in a consistent order. Cases of "self deadlock" when using multiple transactions in one thread are not detected, and a regular timeout exception is thrown instead.
